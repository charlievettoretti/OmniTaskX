# OmniTaskX
**by Charlie Vettoretti**


## Description
This is a full-stack task and time management app that helps you stay organized by tracking your tasks and events. You can create custom categories for better organization and collaborate in shared groups with your friends and family to assign tasks to each other. The built-in AI assistant helps manage your schedule and optimize your daily planning.

## Features
### Task & Event Management
* Create, edit, and delete tasks and events
* Assign due dates, urgency levels, and detailed descriptions
* Mark tasks as Not Started, In Progress, or Completed
* View tasks and events together on a combined calendar

### Categorization
* Create and customize categories with unique colors
* Assign each task or event to a category for better organization
* Filter or group items by category for quick access

### Groups
* Create or join groups with friends, family, or teammates
* View all members in a group
* Add shared tasks visible to all group members
* Assign tasks to specific members and track progress

* Adaptive planning that updates as you complete or reschedule items
